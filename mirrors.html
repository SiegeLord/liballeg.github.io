# Mirrors

* [GitHub project](https://github.com/liballeg) - Some downloads, the git repository and bug tracker are hosted here.

* [TuxFamily project](https://projects.tuxfamily.org/?do=group;name=allegro) - Some downloads and mailing list are hosted here.

* [SourceForge](http://alleg.sourceforge.net/) - The old official web site.

* [Allegro project page at SourceForge](http://sourceforge.net/projects/alleg/) - This is Allegro's project page at SourceForge.

* [Demon Internet](http://www.talula.demon.co.uk/allegro/) - This was the original Allegro web site. Unfortunately it seems to be out of date now. This server hosts only text files.
