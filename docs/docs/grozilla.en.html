Quite a bit of imagery on the conductors list recently has involved "AllegroZilla". I would like to propose an alternative rendering, which I have used once before, which is "'Grozilla". I believe the latter is better for the following reasons:

  1. It looks and sounds like "Godzilla".
  2. Godzilla was a mutated lizard (at least in that really bad movie). 'Grozilla is a mutated Alex, and really big.
  3. It has just enough resemblance to "Allegro", and uses a part of the word that normally gets ignored in abbreviations.
  4. It sounds like "Grow".
  5. It looks and sounds like "Mozilla". We seem to have inherited Mozilla's time scale (or perhaps vice versa).
  6. Godzilla knocks down really big buildings. These buildings (from one possible but incorrect viewpoint) represent obstacles, civilisation, and big buildings. We would like 'Grozilla to do the same. And eat sheep, too.

P.S. Given that we use "AL" so much, why have we not adopted a fat, drunk couch potato for a mascot?
